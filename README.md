# Drone Control Website

Веб-интерфейс для управления дроном через браузер.

## Функционал

- Управление движением дрона (вперед/назад/влево/вправо)
- Управление высотой (взлет/посадка/выше/ниже)
- Отображение телеметрии (высота, скорость, батарея, координаты)
- Подключение к дрону по IP и порту
- Аварийная остановка
- Возврат в точку взлета
- Управление видеопотоком
- Управление с клавиатуры
